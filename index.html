Cask

“To install, drag this icon…” no more!

Homebrew Cask extends Homebrew and brings its elegance, simplicity, and speed to OS X applications and large binaries alike.

It only takes 1 line in your shell to reach 1300 Casks maintained by 500 contributors.

$ brew cask install google-chrome

Install with ease

Your software is just one command away from being ready and raring to go.

Forget all about babysitting the install process step by step, from website to cleanup.

$ ls /opt/homebrew-cask/Caskroom
google-chrome

Unix-savvy, OS X-friendly

Cask does one thing, does it well, and plays nice with others. Apps are kept in their Caskroom under /opt and symlinked to your ~/Applications folder.

Or wherever you like: Cask is fully configurable.

$ brew bundle Caskfile

Automate away

Deployments are a breeze with dotfiles, but only for software that can be built from source.

With Cask, provisioning your apps is just the same: bootstrap a Caskfile and let the machine do the work for you.

$ brew cask create my-cask

Community-driven

Creating new Casks is only a matter of minutes; updating them, of seconds!

And if you feel like improving Cask itself, dive into the codebase and you'll find yourself right at home: Cask is all git and Ruby, just like Homebrew.

Get Cask

$ brew install phinze/cask/brew-cask
